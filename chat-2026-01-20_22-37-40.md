# Phenomenon One Website Build
**Date:** 2026-01-20 22:37

## Summary
Built a minimal portfolio website for phenomenon.one hosted on GitHub Pages.

## Stack
- Plain HTML/CSS (no frameworks)
- Custom font: ABCROMNarrow-Regular.otf
- GitHub Pages hosting
- Custom domain: www.phenomenon.one

## Features Implemented
1. Dark theme with light grey background (#1e1e1e)
2. Custom typography using ABCROMNarrow
3. Breathing circle animation (fades in/out)
4. Theme toggle - click circle to switch between dark/light modes
   - Dark: dark bg, white text, white circle
   - Light: light grey bg (#e5e5e5), black text, blue circle (rgb 0,0,255)
5. Contact mailto link (info@johannesbay.com)
6. Fixed address in bottom left corner (same width as main text - 320px)

## Files
- `index.html` - Single page site with inline CSS
- `CNAME` - Custom domain config (www.phenomenon.one)
- `ABCROMNarrow-Regular.otf` - Custom font
